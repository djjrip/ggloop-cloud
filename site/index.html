Process Scanner

Real-time scanning of running processes against a configurable blocklist. Detects Cheat Engine, aimbots, memory editors, and custom exploit tools.

Active Window Focus

Detect when a player alt-tabs to a known cheat tool, macro software, or overlay injector mid-session. Flag or disconnect instantly.

Zero-Overhead Architecture

Runs in a dedicated background thread. No blocking calls on the game loop. Proven to add <0.1% CPU overhead on both Windows and macOS.

Tamper Detection

Detects debugger attachment, memory patching, and known injection patterns. Triggers configurable callbacks when integrity is violated.
