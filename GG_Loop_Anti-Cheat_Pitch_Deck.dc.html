// Core Capabilities

Everything your game needs.
Nothing it doesn't.

Built from the ground up to be invisible to the player and lethal to cheaters.

🔍

Process Scanner

Real-time scanning of running processes against a configurable blocklist. Detects Cheat Engine, aimbots, memory editors, and custom exploit tools on process launch.

🪟

Active Window Focus Tracking

Detect when a player alt-tabs to a known cheat tool, macro software, or overlay injector mid-session. Flag or disconnect instantly.

⚡

Zero-Overhead Architecture

Runs in a dedicated background thread. No blocking calls on the game loop. Proven to add <0.1% CPU overhead on both Windows and macOS.

🔒

Tamper Detection

Detects debugger attachment, memory patching, and known injection patterns. Triggers configurable callbacks when integrity is violated.

📡

Webhook Reporting

POST violation events to your backend, Discord server, or admin dashboard in real time. Full session context included in every payload.

🧩

Drop-In Integration

Single initialization call. Works with Unity, Unreal, Electron, and raw native apps. TypeScript, C, and C++ bindings included.

// Integration

Integrate in
under 5 minutes.

One import. One initialization call. The SDK handles everything else autonomously — scanning, flagging, and reporting — without touching your game loop.

game-session.ts 
import { AntiCheatSDK } from '@djjrip/anti-cheat-sdk';

// Initialize once on game start
const guard = new AntiCheatSDK({
  sessionId: player.id,
  webhookUrl: 'https://api.ggloop.io/cheat-events',
  scanInterval: 5000,
  blocklist: ['cheatengine', 'artmoney'],
});

guard.on('violation', (event) => {
  // Disconnect, flag, or warn the player
  session.flagPlayer(event.playerId, event.reason);
});

guard.start();
